About This Project

Vehicle Vault is developed as a full-stack Django application with modern UI/UX and real-world features. It includes payment integration with Razorpay, advanced EMI system, insurance management, test drive scheduling, and comprehensive vehicle comparison tools. Built with scalability and security as core priorities.

🛠️ Tech Stack:

Django | Bootstrap 5 | Python | PostgreSQL | Razorpay API | JavaScript | HTML| CSS
